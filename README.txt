Para rodar o programa:
	abrir o terminal na pasta tictactoe-api
	rodar "npm" e "node src/index.js"
	manter aberto, para permanecer com a porta 8080

	abrir o terminal na pasta tictactoe-frontend
	rodar "npm" e "npm start"
	para iniciar o react-app na porta 3000
